# Сортировка жидкостей
## Описание
Программа представляет собой Java-приложение решающее задачу популярной головоломки “сортировка жидкостей”.
Java-проект (Maven), реализующий пошаговое/автоматическое решение головоломки "Сортировка жидкостей"

- Java 21
- Система сборки Maven

## Запуск
1. Клонирование проекта:
   ```bash
   git clone https://github.com/RomanTsitsunov/liquidSort
   cd liquidSort
   ```
2. Запуск проекта:
   ```bash
   mvn clean compile exec:java -Dexec.mainClass="org.example.Main"
   ```
